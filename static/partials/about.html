What is the CRISPR Library Designer?

The development of CRISPR (clustered regularly interspaced short palindromic repeat)-associated Cas9 nuclease has enabled new methods for interrogating gene function. With just a simple alteration of the single-guide RNA (sgRNA) or direct repeat (DR) sequence, one can reprogram a nuclease to target different sites in the genome [1]. Genome-scale CRISPR-Cas9 knockout libraries have emerged as popular tools for unbiased, phenotypic screening [2], but technologies for library design in more specific applications have lagged behind. Our GeCKO library for large scale functional screens, lentiCRISPR V2, was requested from the repository nearly 1,000 times in 2015, yet we still receive nearly daily requests for custom library constructs.

Therefore, we designed the CRISPR Library Designer (CLD): a web application for automated, tissue-specific library design, so scientists can design CRISPR knockout libraries as effortlessly as possible, without sacrificing control. CRISPR Library Designer (CLD), a web application for automated, tissue-specific library design.

How does it work?

Technical Specifications

CLD provides a complete pipeline for the design of custom single guide RNA libraries for use in a wide variety of applications. CLD utilizes fast algorithms to search for genome-wide sgRNA binding sites across an assortment of genes. Furthermore, it integrates tissue expression data from the GTEx Consortium [3] and on-target scores from Doench, et al. and Microsoft Azimuth [4-5] to optimize guide selection for specific tissue-based experiments. When possible, CLD targets functional protein domains based on Pfam data [6], as mutagenesis of functional protein domains leads to a higher proportion of null mutations and enhanced severity of negative selection [7].

To promote scientific accuracy and development of this technology, we have open-sourced all code for the CRISPR Library Designer at the open-source repository GitHub.

For further questions or to report problems, please contact cld@mit.edu.
